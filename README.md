# Sales Performance Analysis

## Objective
To analyze sales data and identify trends, top products, and regional performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Analysis
- Product-wise revenue
- Region-wise distribution
- Monthly sales trend

## Insights
- Printer generates highest revenue
- West region contributes highest revenue
- Sales vary across months

## Conclusion
This analysis helps in understanding sales performance and improving business decisions.
